# Indian-Roadlines
A full-stack web application that aims to visualize the locations of road lines across India.

## Features

- Interactive map
- Spatial database
- Scalable frontend and backend apps
- Reverse proxy server
- Containerized services

## Technologies Used

### Frontend

- Node
- React
- Leaflet
- Bootstrap5

### Backend

- Python
- Django
- GeoDjango
- PostGIS

### API Gateway

- Nginx proxy server
